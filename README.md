# 🌤 Weather Forecast Web App

## 📌 Project Overview

This project is a simple Weather Forecast web application that shows real-time weather information for any city.
Users can search for a city and view the current weather along with a 3-day forecast.

---

## 🚀 Features

* 🌍 Search weather by city name
* 🌡 Display current temperature
* 💧 Show humidity and wind speed
* 📅 3-Day weather forecast
* 🎨 Background color changes based on temperature
* ☁ Weather icons based on condition
* 📱 Fully responsive design

---

## 🛠 Technologies Used

* HTML
* CSS
* JavaScript
* OpenWeather API

---

## 📷 Application Sections

* Search Section – Enter city name to get weather
* Current Weather – Shows temperature, condition, humidity, and wind speed
* 3-Day Forecast – Displays upcoming weather data
